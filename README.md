# Portfolio com Svelte, Javascript e Vite

Este é meu portfolio que eu fiz, parecendo um terminal de verdade.

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/edhlz/portfolio.git
   ```
2. Instale as dependências:
    ```bash
    npm install
    ```
3. Inicie o servidor:
    ```bash
    npm run dev
    ```
